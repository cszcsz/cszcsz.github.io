git常用命令

发表于 2019-08-26 分类于 git 阅读次数：

从github远程克隆项目修改后再推送：

  1. git clone <url> 将项目克隆到本地

  2. git add .  跟踪所有的文件

  3. git commit -m "commit message"  提交文件到本地版本库

  4. git push orgin master   推送到远程仓库

  5. git status  随时查看状态信息
