#### 从github远程克隆项目修改后再推送：

1. `git clone <url>`&emsp;将项目克隆到本地
2. `git add . ` &emsp;跟踪所有的文件
3. `git commit -m "commit message"`  &emsp;提交文件到本地版本库
4. `git push orgin master`  &emsp; 推送到远程仓库 
5. `git status`  &emsp;随时查看状态信息
6. `git checkout -b 分支名`  创建一个新分支并切换到该分支
